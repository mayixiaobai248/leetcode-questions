# 7.22 第一遍
这是一道贪心算法的问题，贪心算法**通过找局部最优解从而找到全剧最优解**

其中很重要的一点：

**贪心算法无套路**

对于这道题分发饼干的问题，每个小孩有个胃口值，饼干也有重量。

那么局部最优解时让这个饼干发挥最大的价值，全局最优解是喂更多的小孩

思路如下：

```
首先排序
for 每一个小孩倒序：
	index = 最后
	for 每一个饼干倒序：
		if index > 0 and 饼干[index] > 小孩：
			result++
			index --
```

有几个点需要注意一下：

>+ 首先一定是for循环遍历小孩胃口，如果遍历的是饼干，这个代码很难写
>+ 先看大饼干和大胃王小孩
